# gmx-socials-js
Integrate Social Networks sharing in your website easily

# How to use:

First, set the socials you want to use and their link:

 <script>
    // Social Name => Link
    var socials = [
      ['Facebook', '#'],
      ['Twitter', '#'],
      ['Linkedin', '#']
    ];
 </script>

Then load gmx-socials.js:

 <script type="text/javascript" src="gmx-socials.js"></script>
